Hacktiv8 Phase 0 Week 1

Reflection

Monday (16/1):

Hari ini saya pertama kali masuk ke grup slack batch 5 program bootcamp Hacktiv8. Setelah memperkenalkan diri, tugas hari ini adalah menyelesaikan kursus di codecademy mengenai HTML & CSS dan membuat halaman HTML sederhana.

Saat melihat profil dan ucapan perkenalan teman-teman seangkatan, ternyata banyak yang mencintai dunia web development dan bersemangat untuk menggapai cita-citanya.. Semangat!


Tuesday (17/1):

Setelah mengenal teman-teman seangkatan, tugas hari ini masih sama yaitu mendalami HTML dan CSS dan menyelesaikan kursus di link codecademy pada hari Senin.

Selain itu, ada juga tugas yang diberikan oleh tim instruktur Hacktiv8, yakni menghias HTML dengan CSS. Saya belajar bagaimana menggunakan selector class untuk menghias HTML dengan CSS. Seru!


Wednesday (18/1):

Hari ini judulnya mempelajari CSS lebih dalam lagi. Saya masih disuruh untuk menyelesaikan kursus di codecademy.

Saya mendapat banyak pengetahuan tentang penggunaan CSS properties sehingga bisa menghias HTML dengan lebih cantik lagi.


Thursday (19/1):

Akhirnya saya mendapatkan materi tentang penggunaan git dan github sebagai sebuah sistem revisi untuk programmer. Penggunaan keduanya penting jika ingin sukses sebagai seorang profesional di dunia pengembangan aplikasi web atau sejenisnya.

Selain itu, saya juga belajar menggunakan markdown dan membuat file readme di github dengan mengaplikasikan pengetahuan tentang markdown yang baru didapat. Keseruan lain adalah membuat branch baru dan berlatih bagaimana git berguna bagi karir seorang web developer seperti saya.


Friday (20/1):

Akhirnya sampai juga ke akhir pekan. Dimana tidak akan ada tugas lagi sampai senin besok, hehe. Di minggu ini, selain belajar materi tentang perencanaan, proses dan manajemen pembuatan web, saya juga membuat website ini!

Sekedar info, website ini dihosting gratis menggunakan fitur github, yakni pages. Cara kerjanya, intinya file index.html yang di upload di github akan langsung bisa diakses karena dihosting secara otomatis. Lebih jauh, website seperti ini dikenal dengan nama static website.


Saturday & Sunday (21-22/1):

Setelah menjalani minggu pertama di fase 0 program bootcamp Hacktiv8, saya mengingat kembali pengetahuan tentang HTML dan CSS yang dulu pernah dipelajari. Hanya saja, secara pribadi ada beberapa kesulitan yang saya alami.

Kesulitan ini bukan pada bagian teknis, tapi lebih kepada kesehatan yang menghambat saya dalam belajar materi dan mengerjakan tugas. Akhirnya, banyak tugas yang belum saya kerjakan.

Sehari sebelum Senin (16/1), saya harus dioperasi wasir atau ambeyen. Rasanya sih tidak apa-apa karena masih dibius lokal, tapi kira-kira dua belas jam pasca operasi, saya harus berjuang menahan rasa sakit.

Pada saat itu, saya benar-benar tidak bisa kemana-mana selain berbaring di kasur. Untuk buang air, semua serba otomatis (organ depan dipakai kateter).

Untuk buang air besar, saya hanya mengeluarkan cairan karena hanya diberi air putih dan susu formula saja. Hal ini dilakukan untuk menghindari rasa nyeri dari kotoran yang keras jika memakan makanan berat.

Selama tiga hari saya di rumah sakit sehingga boleh pulang. Namun, begitu sampai rumah pun belum bisa beraktivitas seperti biasa. Rasa sakit masih ada dan untuk buang air besar pun sulit dan nyeri.

Hal itulah yang menghambat saya untuk mempelajari materi dengan lebih banyak di minggu kesatu ini. Untungnya, rasa sakit tidak berlama-lama dan saya bisa menyelesaikan tugas di minggu selanjutnya dengan lebih baik lagi.

Semoga saya bisa lolos ke fase selanjutnya dan kesehatan saya tidak menjadi halangan lagi. Semoga kita semua diberikan kesehatan dan perlindungan Allah SWT. Amin..
